## How to Play

1. **You play as White** (bottom of the board)
2. Click a piece to select it
3. Click a destination square to move
4. The computer will respond automatically
5. Use the theme selector to change board appearance
6. View analysis panel on the right for move insights

## Controls

- **New Game**: Start a fresh game
- **Undo**: Take back your last move (and computer's response)
- **Theme Button**: Switch between board themes
- **Training Tips**: Click the book icon for chess wisdom
